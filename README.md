# UI5 Web Components for React

This repository hosts the solution to the SAP Tutorial Mission: [Build a Single Page Application Using UI5 Web Components for React](https://developers.sap.com/mission.react-spa.html).

## Getting Started

- To run the app, issue command below:

```shell
> npm install
```

- Then

```shell
> npm start
```

## Generate App

- Create the app from UI5 Web Components template

```shell
> npx create-react-app . --template @ui5/cra-template-webcomponents-react
```
